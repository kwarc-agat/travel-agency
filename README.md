# Travel Agency

Single page application for travel agency. Presenting currently offered tours and provides contact form for customers.

## Features

- grid with offered tours
- contact form
- Google Map
- smooth navigation
- modern design

## Technologies

- Bootstrap 5
- Material Design
- Google API
- HTML, CSS
